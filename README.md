# fanqienovel-downloader

下载番茄小说，通过Python实现。

## 使用方法

1. 输入id以下载，如：
**fanqienovel.com/page/7276384138653862966** 中的 **7276384138653862966**
2. 输入gx以更新，读取 record.json 中的id进行更新。

## 关于磁盘占用的说明

为防止程序因意外关闭，每下载一章便保存一次，使用磁盘占用相对较多(3+MB/S)。
